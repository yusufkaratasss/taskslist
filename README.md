## Açıklama

Bu proje, kullanıcı listesini yönetmek için bir uygulamadır. İşte projenin temel özellikleri:

- Kullanıcı ekleme
- Kullanıcı düzenleme
- Kullanıcı silme
- Görev ekleme
- Görev silme
- Görev güncelleme
- Görev tipi ekleme
- Görev tipi silme
- Görev tipi güncelleme

## Proje İndirme ve Kurulum

1. 
    ```bash
    npm init
    ```

2. 
    ```bash
    npm install
    ```

3. 
    ```bash
    npm install cors
    ```

4. 
    ```bash
    npm install express
    ```

5. 
    ```bash
    npm install mongodb
    ```

6. 
    ```bash
    npm install jsonwebtoken 
    ```

7. 
    ```bash
    npm install  bcrypt 
    ```

8. 
    ```bash
    npm install nodemon 
    ```
   - nodemon çalışmazsa
    ```bash
    export PATH=$PATH:$(npm -g bin)
    ```
